## Instructions

Analyze the task and generate a sequence of commands to accomplish it.

**CRITICAL: Understanding Object Coordinates**
- All object positions represent the CENTER point of the object
- When you see an object at position [x, y, z] with height h, this means:
  - The object's center is at height z
  - The object's bottom is at height z - h/2
  - The object's top is at height z + h/2

**Stacking Rules**
When stacking object A on top of object B:
1. Find object B's center position [x_b, y_b, z_b]
2. Use the SAME x and y coordinates: [x_b, y_b]
3. Calculate the new z coordinate:
   - Object B's top surface is at: z_b + height_b/2
   - Object A's center should be at: z_b + height_b/2 + height_a/2
   - Simplified formula: z_new = z_b + (height_b + height_a)/2
4. Place object A at position [x_b, y_b, z_new]

**Example Calculation:**
- Object B at [0.25, 0.45, 0.05] with height 0.1
- Object A (being placed) has height 0.1
- Place position: [0.25, 0.45, 0.05 + (0.1 + 0.1)/2] = [0.25, 0.45, 0.15]

Task: Place the red cube on top of the blue cube.

- blue_cube at [0.5, 0.5, 0.05] with dimensions [0.1, 0.1, 0.1]

Solution:
```json
{{
  "commands": [
    {{"action": "pick_up", "object": "red_cube"}},

```
Explanation:

- Place red cube at blue cube's center position with adjusted height: [0.5, 0.5, 0.15]

### Example 2: Sequential Stacking
Task: Stack the green cube on the red cube, then stack the blue cube on top of the green cube.
Objects:
- green_cube at [0.2, 0.3, 0.05] with dimensions [0.1, 0.1, 0.1]
- red_cube at [0.4, 0.4, 0.05] with dimensions [0.1, 0.1, 0.1]
- blue_cube at [0.6, 0.5, 0.05] with dimensions [0.1, 0.1, 0.1]

Solution:
```json
{{
  "commands": [
    {{"action": "pick_up", "object": "green_cube"}},
    {{"action": "place", "position": [0.4, 0.4, 0.15]}},
    {{"action": "pick_up", "object": "blue_cube"}},
    {{"action": "place", "position": [0.4, 0.4, 0.25]}}
  ]
}}
```
Explanation:
- Step 1: Place green on red
  - Red cube center: [0.4, 0.4, 0.05]
  - Use red's x, y: [0.4, 0.4]
  - Calculate z: 0.05 + (0.1 + 0.1)/2 = 0.15
  - Place green at red's center position: [0.4, 0.4, 0.15]
- Step 2: Place blue on green
  - Green cube is now at [0.4, 0.4, 0.15]
  - Use green's x, y: [0.4, 0.4]
  - Calculate z: 0.15 + (0.1 + 0.1)/2 = 0.25
  - Place blue at green's center position: [0.4, 0.4, 0.25]
